# DexBurger RS 

#### Run this before anything:

`npm install`

#### Local dev server

`npm start`<br />
access http://localhost:3000/

#### Build to production

`npm run build` <br />
Production content will be generated into /dist folder

#### Running into production

Copy dist folder content into your server and run
`node ./bin/www` <br />
